Espresso Market Pricing Analysis

Project Summary

This research investigates pricing and competition in the UK espresso machine sector using primary data from Amazon UK. I built a custom tool to identify price drivers and assess market concentration within a digital marketplace.

Technical Implementation

I used a VBA script to scrape 303 unique observations and utilised Power Query to clean and standardise the raw metadata. My analysis involved estimating hedonic pricing models in Stata while using robust standard errors to ensure reliable statistical results.

Analytical Findings

Concentration analysis reveals that purchasing activity is concentrated among elite manufacturers with a review based C4 ratio of 70.20% and an HHI of 1785.54. Regression results show that machine weight is the strongest predictor of price as a one kilogram increase leads to a 13.8% price premium. These findings suggest that technical differentiation and brand reputation are the primary drivers of market value.

Tools Used

The project was completed using VBA for browser automation and Power Query for data transformation. I also used Microsoft Excel for dataset management and Stata for econometric modelling and diagnostic testing.
